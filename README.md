# Netfleet

This is a simple symfony movies/tv series micro-api project.

## Installation

### Requirements

- PHP 8
- Composer
- Symfony
- MySQL

### Install

- Clone the project
- Install the vendors
- Create the database

#### Clone the project

    git clone git://github.com/evalSymf.git

#### Install the vendors

        cd netfleet_api
        composer install

#### Create the database

            doctrine:database:create
